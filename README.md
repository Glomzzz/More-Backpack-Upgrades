# MoreBackpackUpgrades (`mobackup`)

Port to NeoForge for Minecraft `1.21.1`.


## Quickstart

1. Clone the repository:
    - `git clone https://github.com/Glomzzz/More-Backpack-Upgrades.git`
    - `cd More-Backpack-Upgrades`

2. Build:
    - `./gradlew build`  
      The mod JAR will be produced in `build/libs/` (or `build/jars/libs/` depending on task configuration).
